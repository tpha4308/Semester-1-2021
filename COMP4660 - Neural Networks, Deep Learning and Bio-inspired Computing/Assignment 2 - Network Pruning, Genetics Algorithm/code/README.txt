Genetic Algorithms and Network Pruning in Improving Deceit Detection Using Neural Networks and Pupillary Responses

-------------- Dataset used 

subjective_belief_observers_features_labels.csv


-------------- PLEASE RUN IN THE ORDER BELOW


To set up the neural net object with training functions 
Run model.py

To initialise parameters for a baseline model
Run initialisation.py

To inspect data
Run data-description.py

To check for the best baseline hyper parameters
Run baseline-hyperparameters.py

To run genetics algorithm to find the best subset of features
Run genetic-algo.py

To run classifications for section 4.1 and 4.2 (needs to be run AFTER genetic-algo.py)
Run classification.py

To perform hidden unit pruning 
Run pruning.py


-------------- Other existing files 

og-model is initial model parameters generated by initilisation.py

subsetGA.npy is the feature subset generated by genetic-algo.py

save-model is a folder to save models' parameters generated in classification.py

relevant-papers is a folder contains the relevant papers as mentioned below
